# 🇿🇦 SlotZar

> **WhatsApp-native booking + payment collection for South African service businesses.**

SlotZar helps Cape Town salons, barbers, tutors, and service businesses take bookings AND collect payment upfront — all via WhatsApp. No more no-shows. No more chasing payments.

---

## 🎯 The Problem

South African service businesses manage bookings on WhatsApp, but:
- **No-shows** cost R150–R500 per empty slot
- **Manual admin** wastes 2+ hours per day
- **Invoices go unpaid** — nearly half of SA SME owners chase payments for up to 2 months

Existing tools (Bookly, SimplyBook, WhatsBookings) generate invoices but **don't collect payment at booking time**. That's the gap SlotZar fills.

---

## ✅ What SlotZar Does

| Feature | Description |
|---|---|
| 📅 Booking Page | Branded page at `slotzar.co.za/yourname` — clients book 24/7 |
| 💬 WhatsApp Reminders | Auto-reminder 24h before appointment via WhatsApp |
| 💳 Payment at Booking | Yoco/PayFast link sent at booking — pay upfront or no slot |
| 🧾 Auto Invoicing | Professional invoice generated + sent automatically |
| 📆 Calendar Sync | Every booking syncs to Google Calendar |
| 📢 Client Broadcasts | Send WhatsApp promos to fill quiet slots |

## 🛠️ Tech Stack

- **Frontend:** Next.js 14 + Tailwind CSS
- **Backend:** Node.js / Supabase
- **Payments:** Yoco + PayFast
- **Messaging:** WhatsApp Business API (Twilio)
- **Auth:** Supabase Auth
